Courses

Novice-level:
MDN Web Docs - Learn Web Development:

MDN Web Docs by Mozilla provides comprehensive and beginner-friendly documentation on web development. It covers HTML, CSS, JavaScript, and more. The tutorials are well-structured, and MDN is a reliable resource for web development.

freeCodeCamp:

freeCodeCamp offers a full curriculum for web development, including HTML, CSS, JavaScript, and various frameworks. It's interactive, project-based, and provides hands-on coding challenges. The platform also offers a responsive community for support.

Codecademy - Web Development path:

Codecademy has a free version that includes an interactive web development path. It covers HTML, CSS, JavaScript, and more. The hands-on coding exercises make it a great choice for many beginners who prefer learning by actually doing.

Intermediate-level:
Coursera - Web Design for Everybody:

Web Design for Everybody is offered by the University of Michigan on Coursera. It covers code such as HTML5, CSS3, and JavaScript, along with responsive design principles. While the specialization is free, there might be a fee to recieve the certification.

edX - HTML, CSS, and JavaScript for Web Developers:

This is a course offered by the University of Hong Kong on edX. It covers the basics of HTML, CSS, and JavaScript and then progresses to more advanced topics. You can audit the course for free, with the option to pay for a certificate.

Udemy - The Complete Web Developer Course 2.0:

The Complete Web Developer Course 2.0 by Rob Percival on Udemy covers a wide range of web development technologies. While Udemy courses are often paid, this one is frequently discounted, and you might be able to find it for free during promotions.
